# My-Spring Project

## 🌟 概要
Spring Boot を使ったバックエンド + React を使ったフロントエンド構成のWebアプリケーションです。

## 🛠️ 技術スタック
- Java / Spring Boot
- javasprict/Node.js / React
- Docker / PostgreSQL
- GitHub Actions（CI/CD用）

## 📌 機能（一部）
- ログイン／新規登録
- RESTful API連携
- フロント〜バック統合通信
- DB接続とCRUD操作

## 🚀 起動方法
```bash
docker-compose up --build
